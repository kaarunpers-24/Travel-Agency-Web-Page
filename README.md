**Travel Agency Webpag**
This is a simple travel agency webpage I built while learning HTML. The goal of this project was to practice writing clean, well-structured code and understanding how real webpages are organized.
 About the Project
I created this page as part of the freeCodeCamp Responsive Web Design curriculum. It helped me understand semantic HTML, links, images, lists, and how small details matter when building real-world layouts.

**What I Learned**
•	How to structure a webpage using semantic HTML  
•	Proper use of lists, links, and figures  
•	Writing HTML that follows the exact requirements  
•	Paying attention to small details and accessibility  

**Tech Used**
 HTML5
